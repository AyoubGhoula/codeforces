# 🧠 Codeforces Solutions

## 📌 Description

This repository contains my solutions to problems from Codeforces.
The goal of this repo is to track my progress, improve problem-solving skills, and build consistency in competitive programming.

---

## 🎯 Objectives

* Practice data structures and algorithms
* Improve problem-solving speed and accuracy
* Prepare for coding interviews and contests
* Track progress over time

---


## 🛠️ Technologies Used

* Language: Python 🐍 and C++ 
* Concepts:

  * Arrays
  * Strings
  * Sorting
  * Greedy algorithms
  * Dynamic programming
  * Graphs

---

## ▶️ How to Run

```bash
python solution.py
```

Some problems require input from standard input:

```bash
python solution.py < input.txt
```
